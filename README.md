# CSCB20 Web App

Welcome to the UTSCEats Web App repository! This CSCB20 project showcases an intuitive and efficient Minimal Viable Product (MVP) of a web application designed specifically for students. 

## Features

- Account creation: Students can effortlessly create their accounts to get started.
- Food ordering: Explore and order delicious food from various campus vendors.
- Convenient order pickups: Easily pick up your orders at designated locations.
- Vendor portal: Vendors can efficiently manage menus and handle incoming orders.

## Technologies Used

- Python
- Flask
- SQLAlchemy
- Bootstrap
- HTML/CSS/JS

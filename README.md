# CSCB20 Web App

Welcome to the UTSCEats Web App repository! This CSCB20 project showcases an intuitive and efficient Minimal Viable Product (MVP) of a web application designed specifically for students at the University of Toronto Scarborough. 

## Features

- Account creation: Students can effortlessly create their accounts to get started.
- Food ordering: Explore and order delicious food from various campus vendors.
- Convenient order pickups: Easily pick up your orders at designated locations.
- Vendor portal: Vendors can efficiently manage menus and handle incoming orders.
- User-friendly UI/UX: Our web app prioritizes a seamless and intuitive user experience. With a clean and modern interface, users can effortlessly navigate through the application, ensuring a smooth and enjoyable interaction. Users can quickly accomplish tasks, such as creating accounts, placing orders, managing preferences, and more. 

## Technologies Used

- Python
- Flask
- SQLAlchemy
- Bootstrap
- HTML/CSS/JS
